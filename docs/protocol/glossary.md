`CHIPS` - game token and the name of the poker project

`DCV` - Deck Creating Vendor, the Dealer node

`BVV` - Blinding Value Vendor, the component that helps securely shuffle the deck

`Cashier` or `Notary` nodes - Trusted nodes in the network that are elcted and chosen by the community. The set of trusted nodes at the moment are [here](../../privatebet/config/cashier_nodes.json).
